In a world where people work mainly on computers, we still communicate with word / PDF document, which are no more than digitized paper sheets.

This is complete nonsense.

Software developers and geeks already solved this with documentation tools like [sphinx](https://www.sphinx-doc.org/en/master/usage/markdown.html), [vitepress](https://vitepress.dev/), [jekyll](https://jekyllrb.com/), [hugo](https://gohugo.io/)...

All these are good when you want to share code. But what about what other disciplines need: graphs, bibliographic references, complex figures, 3D objects, glossaries ? And if you still want to print you work in a beautiful format ?

Unfortunately, the perfect tool does not exist –*yet*.\
ssphd is our humble attempt to make a step closer to this ideal.
